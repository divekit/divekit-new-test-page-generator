# divekit-new-test-page-generator

## Nutzung

* Mit `node bin/generate-report` lässt sich der Testseitengenerator lokal ausführen. Dazu sollte vorher die vom Report-Mapper generierte Datei `unified.xml` im `target` Ordner abgelegt werden.

* Der Output des Testseitengenerator befindet sich im `public`-Ordner
